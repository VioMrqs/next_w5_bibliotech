# BilioTech Project

## Specs

This project aims at learning to use React js.

The goal was to 
- Parse & gather books data from a json file,
- Display them nicely (cards),
- Add books to a wishlist or favorites
- Filter them according to those 2 filters
- Implement a search bar
- & implement a little bit of scss
